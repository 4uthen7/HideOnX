Hide On X 🛡️
X (Twitter) のプロフィール情報を隠蔽し、最小限のアカウントスイッチャーを提供。

📌 概要
Hide On X は、X（旧Twitter）のサイドバー左下に表示されるユーザー名・ID・アイコンを非表示にし、プライバシーを保護するためのブラウザ拡張機能です。 単に削除するだけでなく、複数アカウントを利用するユーザーのために、アカウント切り替えメニューを呼び出す専用ボタンを新たに実装します。

✨ 主な機能
👤 プロフィール情報の隠蔽: ユーザー名、ID、アバターを完全に非表示にします。

🔄 アカウント切り替えボタン: 視認性の高い上下矢印アイコン（SVG）を配置し、スムーズなアカウント操作を維持します。

🎯 XのUIへの最適化: ダークモードを含むXのデザインに馴染む、シンプルで直感的なボタンデザインを採用。

⚡ 動的な監視機能: MutationObserver により、ページ遷移やアカウント切り替え後も即座にスタイルを適用。

🛠 技術スタック
コア: JavaScript (Vanilla JS)

仕様: Web Extensions API (Manifest V3)

監視: MutationObserver によるDOM変更検知

デザイン: インラインCSS（JSによる直接制御）

📦 インストール方法
本リポジトリをZIP形式でダウンロードし、任意の場所に解凍します。

ブラウザで拡張機能管理画面を開きます（例: chrome://extensions/）。

画面右上の「デベロッパー モード」を有効にします。

「パッケージ化されていない拡張機能を読み込む」を選択し、解凍したフォルダを指定します。

⚠️ 注意事項
仕様変更への対応: Xの要素ID（data-testid等）に依存しているため、X側のアップデートにより動作しなくなる場合があります。

プライバシー: 本拡張機能はオフラインで動作し、ユーザーデータを外部サーバーへ送信することはありません。🔒

互換性: 他のUIカスタマイズ系拡張機能と競合し、表示が崩れる可能性があります。
