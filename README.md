# HideOnX
X (Twitter) のプロフィール情報を隠し、プライバシーを守るミニマルなアカウントスイッチャー
概要
Hide On X は、X（旧Twitter）のサイドバー左下に表示されるユーザー名・ID・プロフィールアイコンを瞬時に非表示にし、代わりに洗練された「アカウント切り替え専用ボタン」を実装するブラウザ拡張機能です。

カフェなどの公共の場でのブラウジング、画面共有会議、動画配信など、意図しないアカウント情報の露出を防ぎたいシーンに最適です。

主な機能
プロフィールパーティクルの自動削除: サイドバーの個人情報を完全に隠蔽します。

専用スイッチャーボタンの生成: 既存の情報を隠しながら、アカウント切り替えメニューを呼び出すための「上下矢印アイコン」を新設します。

ネイティブUIとの親和性: 生成されるボタンはXのダークモードやUIデザインに馴染むよう設計されています。

高速な動作: MutationObserver を利用し、ページの読み込みやアカウントの切り替え後も即座に適用されます。

技術スタック
コア: JavaScript (Vanilla JS)

API: Web Extensions API (Manifest V3)

手法: DOM操作 / MutationObserver による動的要素監視

デザイン: インラインCSSによるスタイリング（SVGアイコン採用）

インストール方法
本リポジトリをダウンロード（Code > Download ZIP）し、解凍します。

ブラウザで拡張機能管理画面を開きます（例: Chromeの場合は chrome://extensions/）。

「デベロッパー モード」を有効にします。

「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

使用上の注意
Xの仕様変更: 本拡張機能はXの特定の要素（data-testid）を対象に動作します。X側のアップデートにより、一時的に動作しなくなる可能性があります。

プライバシー: 本拡張機能はユーザーデータを収集・送信することはありません。すべての処理はローカルのブラウザ上で行われます。
